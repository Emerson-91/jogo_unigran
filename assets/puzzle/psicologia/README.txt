Imagens do jogo da memória de Psicologia.

Necessário adicionar os arquivos PNG abaixo (nomes exatos):
- gestalt.png
- psicanalise.png
- tcc.png
- social.png

Cada imagem representa uma abordagem/conceito de Psicologia.
Resolução recomendada: até ~400x300 (será reduzida para caber na carta 200x80 com escala, preferir aspecto horizontal ou quadrado simples).
Fundo transparente ou sólido claro.

Fallback: se a imagem não carregar, o nome (Gestalt, Psicanálise, TCC, Social) aparece centralizado na carta.
